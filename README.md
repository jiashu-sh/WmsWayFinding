# A*简单寻路算法以及wms地图模块

整个项目基于C#，“Ai/WayfindingAStar.cs” （命名已修正）是A*算法的核心类，其他是一些winform窗体之类的，为了能展示寻路过程，做了LocationMgt/FormLaneMgt.cs 这个窗体。

整个项目比较简单，主要是将整个form分为若干个方格，可以右键添加障碍（不可走）、设置起点和终点，开始运行后即绘制寻路轨迹。

增加了产品维护，但不属于这个项目相关内容，已经复制为另一个项目。
